Computer Architecture Quiz

1. What is microcode?

A low-level instruction used by processors.
A high-level programming language.
A type of cache memory.
An operating system function.

Correct Answer: A low-level instruction used by processors.

2. What does the ISA (Instruction Set Architecture) define?

The hardware components of a computer.
The number of processor cores.
The set of instructions a processor can execute.
The memory hierarchy structure.

Correct Answer: The set of instructions a processor can execute.

3. Which hazard occurs when one instruction depends on the result of a previous instruction?

Control hazard
Structural hazard
Data hazard
Branch hazard

Correct Answer: Data hazard

4. What is a superscalar processor?

A processor that can execute multiple instructions simultaneously.
A processor with a single execution unit.
A type of memory management unit.
A processor that uses microcode exclusively.

Correct Answer: A processor that can execute multiple instructions simultaneously.

5. What does the term "branch prediction" refer to?

Predicting the execution order of instructions.
Guessing the outcome of a conditional branch.
Predicting the next instruction in the sequence.
Predicting the memory address of data.

Correct Answer: Guessing the outcome of a conditional branch.

6. What is a pipeline in computer architecture?

A single stage that handles all instruction types.
A sequence of stages that process instructions in parallel.
A way to connect multiple processors.
A technique used only for data storage.

Correct Answer: A sequence of stages that process instructions in parallel.

7. What is the purpose of the cache memory?

To provide fast access to frequently used data.
To store all data permanently.
To hold the operating system.
To process complex calculations.

Correct Answer: To provide fast access to frequently used data.

8. What does "hazard" in pipelining refer to?

An issue that prevents instructions from executing simultaneously.
A method of improving CPU performance.
A type of memory used in pipelining.
A stage in instruction processing.

Correct Answer: An issue that prevents instructions from executing simultaneously.

9. Which of the following is a type of data hazard?

Control hazard
Write-after-write hazard
Branch hazard
Instruction hazard

Correct Answer: Write-after-write hazard

10. What is the function of the control unit in a CPU?

It stores instructions for execution.
It processes data.
It coordinates the execution of instructions.
It manages the input and output operations.

Correct Answer: It coordinates the execution of instructions.

11. What is an example of a data hazard in pipelining?

Read-after-write hazard
Write-after-write hazard
Branch hazard
Control hazard

Correct Answer: Read-after-write hazard

12. Which of the following is true about a superscalar processor?

It can only execute one instruction per clock cycle.
It can execute multiple instructions simultaneously.
It uses a single execution unit.
It does not support pipelining.

Correct Answer: It can execute multiple instructions simultaneously.

13. What is the role of the cache memory?

To hold data that is rarely used.
To speed up access to frequently used data.
To store the operating system.
To manage input/output operations.

Correct Answer: To speed up access to frequently used data.

14. What does branch prediction help to avoid in a pipeline?

Data hazards
Control hazards
Structural hazards
Memory hazards

Correct Answer: Control hazards

15. In the context of pipelining, what is a structural hazard?

A hazard caused by a data dependency between instructions.
A hazard caused by a limited number of execution units.
A hazard caused by branch instructions.
A hazard caused by memory access.

Correct Answer: A hazard caused by a limited number of execution units.

16. What does the "instruction set architecture" (ISA) of a processor define?

The hardware components of the processor.
The set of instructions a processor can execute.
The memory hierarchy.
The processor's cache size.

Correct Answer: The set of instructions a processor can execute.

17. What is VLIW (Very Long Instruction Word) architecture?

A processor architecture that allows for simultaneous execution of multiple independent instructions.
A processor architecture with a single instruction per cycle.
A memory management technique.
A type of pipelining.

Correct Answer: A processor architecture that allows for simultaneous execution of multiple independent instructions.

18. What is a control hazard?

A hazard caused by branch or jump instructions.
A hazard caused by a memory access conflict.
A hazard caused by an insufficient number of execution units.
A hazard caused by data dependencies.

Correct Answer: A hazard caused by branch or jump instructions.

19. What is a pipelining hazard in a CPU?

A problem that causes a delay in instruction processing.
A conflict between two memory accesses.
A delay caused by waiting for an instruction to execute.
A conflict due to insufficient processor cores.

Correct Answer: A problem that causes a delay in instruction processing.

20. What does a structural hazard occur in a pipeline?

A lack of available resources to execute an instruction.
A conflict between instructions needing the same resources.
A data conflict between instructions.
A conflict caused by a jump instruction.

Correct Answer: A conflict between instructions needing the same resources.

21. What is the primary purpose of a TLB (Translation Lookaside Buffer)?

To speed up the translation of virtual addresses to physical addresses.
To store data temporarily.
To increase the size of cache memory.
To manage input and output devices.

Correct Answer: To speed up the translation of virtual addresses to physical addresses.

22. Which of the following is an example of a "read-after-write" data hazard?

Instruction 1 writes to a register, and Instruction 2 reads from the same register.
Instruction 1 writes to memory, and Instruction 2 reads from memory.
Instruction 1 reads from a register, and Instruction 2 writes to the same register.
Instruction 1 writes to memory, and Instruction 2 writes to the same memory address.

Correct Answer: Instruction 1 writes to a register, and Instruction 2 reads from the same register.

23. What is the primary difference between a RISC and CISC processor?

RISC uses fewer instructions, and CISC uses more complex instructions.
RISC has more complex instructions than CISC.
CISC uses fewer instructions, and RISC uses more complex instructions.
RISC and CISC are identical in their instruction sets.

Correct Answer: RISC uses fewer instructions, and CISC uses more complex instructions.

24. In a superscalar architecture, what is the main function of the issue stage?

To execute instructions in parallel.
To decode and prepare instructions for execution.
To handle instruction scheduling.
To fetch instructions from memory.

Correct Answer: To decode and prepare instructions for execution.

25. Which of the following is a type of pipeline hazard?

Data hazard
Control hazard
Structural hazard
All of the above

Correct Answer: All of the above

26. What does the term "out-of-order execution" refer to in modern CPUs?

Executing instructions in the order they appear in the program.
Executing instructions based on available resources, regardless of their original order.
Storing instructions in a buffer before executing them.
Repeating instructions to ensure correctness.

Correct Answer: Executing instructions based on available resources, regardless of their original order.

27. What is the main purpose of branch prediction in modern processors?

To reduce memory access time.
To speed up the execution of instructions by guessing the direction of branches.
To handle data hazards.
To allocate resources efficiently.

Correct Answer: To speed up the execution of instructions by guessing the direction of branches.

28. In a pipelined CPU, which type of hazard occurs when two instructions need the same hardware resource?

Data hazard
Control hazard
Structural hazard
Memory hazard

Correct Answer: Structural hazard

29. What is the purpose of using multiple execution units in a superscalar processor?

To execute different types of instructions simultaneously.
To speed up memory access.
To reduce the need for pipelining.
To handle larger instruction sets.

Correct Answer: To execute different types of instructions simultaneously.

30. What does the "fetch" stage of a pipeline do?

It decodes instructions.
It fetches instructions from memory.
It executes the instructions.
It stores results in memory.

Correct Answer: It fetches instructions from memory.

31. What is a cache miss?

When the data is found in the cache.
When the data is not found in the cache and needs to be fetched from memory.
When the cache is full.
When the cache is being updated.

Correct Answer: When the data is not found in the cache and needs to be fetched from memory.

32. Which type of processor architecture is designed to execute a small, simple set of instructions?

CISC
RISC
VLIW
Superscalar

Correct Answer: RISC

33. What is the main benefit of a VLIW (Very Long Instruction Word) architecture?

It executes multiple instructions in parallel in a single clock cycle.

It allows out-of-order execution of instructions.

Correct Answer: It executes multiple instructions in parallel in a single clock cycle.

34. What is the function of the instruction decode stage in a pipelined processor?

It fetches the instruction from memory.
It decodes the instruction to determine the operation.
It executes the instruction.

Correct Answer: It decodes the instruction to determine the operation.

35. What is the purpose of instruction pipelining in modern processors?

To execute multiple instructions in parallel.
To increase the clock speed of the processor.
To break down instructions into smaller tasks.
To execute instructions faster by overlapping different stages of processing.

Correct Answer: To execute instructions faster by overlapping different stages of processing.

36. In a superscalar processor, what is the role of the issue stage?

To fetch instructions from memory.
To dispatch instructions to execution units.

37. What is a key advantage of out-of-order execution in processors?

It minimizes the impact of pipeline hazards.
It speeds up the execution of instructions.

Correct Answer: It minimizes the impact of pipeline hazards.

38. What is a structural hazard in a pipeline?

When two instructions try to access the same resource simultaneously.
When data dependencies between instructions cause delays.

39. What is a control hazard in pipelined processors?

When an instruction depends on the result of a previous instruction.

40. What is the primary purpose of branch prediction in processors?

To increase the number of instructions executed per cycle.

41. What does the term "cache hit" mean?

When the cache is empty.

42. Which of the following is a primary characteristic of a RISC processor?

43. What is the advantage of using out-of-order execution in a CPU?

45. What is a pipeline hazard?

47. What is the role of the "fetch" stage in a pipeline?
